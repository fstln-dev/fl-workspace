### 4.1 API 列表

|------|------|------|------|
| 获取列表 | GET | /api/v1/items | 分页查询 |

| 更新 | PUT | /api/v1/items/:id | 更新数据 |
| 删除 | DELETE | /api/v1/items/:id | 删除数据 |

### 4.2 接口详情

#### GET /api/v1/items

**请求参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| size | int | 否 | 每页数量，默认 20 |

**响应示例**:

```json
{
  "code": 0,
  "data": {
    "items": [],
    "total": 100,
    "page": 1,
    "size": 20
  }
}
```
